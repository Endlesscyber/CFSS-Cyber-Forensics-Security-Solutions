# Cyber Security Internship Progress

## Week 1: Advanced Reconnaissance and Footprinting

During Week 1, I learned and practiced Advanced Reconnaissance and Footprinting techniques. This included gathering information about targets using passive and active reconnaissance methods, understanding OSINT techniques, and identifying potential attack surfaces.

## Week 2: Web and Client-Side Attacks

In Week 2, I focused on Web and Client-Side Attacks. I explored common web vulnerabilities, client-side attack vectors, and techniques used to identify and assess security weaknesses in web applications.

## Week 3: System Hacking and Wireless Security

During Week 3, I studied System Hacking and Wireless Security. This involved learning about system exploitation techniques, privilege escalation concepts, password attacks, wireless network security, and methods used to secure wireless environments.

## Week 4: Post Exploitation and Hardening

In the final week, I worked on Post Exploitation and Hardening. I learned how attackers maintain access after exploitation, techniques for assessing the impact of a compromise, and best practices for system hardening to improve overall security.

## Conclusion

Throughout these four weeks, I gained practical knowledge of various cybersecurity domains, including reconnaissance, web security, system security, wireless security, post-exploitation techniques, and defensive hardening measures.
